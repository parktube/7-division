# CAD Voxel Skill - Section Definitions

## Sections

### tools
- **title**: MCP 도구 사용법
- **impact**: critical
- **description**: CAD MCP 도구 (glob, read, edit, write, lsp, bash) 사용 패턴

### functions
- **title**: CAD 함수 레퍼런스
- **impact**: critical
- **description**: 도형 생성, 변환, 스타일, 그룹 함수 설명

### zorder
- **title**: Z-Order 관리
- **impact**: high
- **description**: 복셀 아트에서 올바른 깊이 표현을 위한 z-order 패턴

### coords
- **title**: 좌표 시스템
- **impact**: high
- **description**: 그룹/로컬/월드 좌표 관리 패턴

### syntax
- **title**: 문법 규칙
- **impact**: medium
- **description**: CAD 코드 작성 시 지켜야 할 문법 규칙 (ES5)

### workflow
- **title**: 워크플로우
- **impact**: medium
- **description**: 설계-구현-검증 워크플로우
