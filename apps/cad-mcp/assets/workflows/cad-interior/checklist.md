# CAD Interior Design - Validation Checklist

각 Phase 완료 전 확인해야 할 항목들입니다.
이것은 "통과/실패" 체크가 아니라 **"충분히 탐색했는지"** 확인입니다.

---

## Phase 1: Discovery 체크리스트

### 필수 확인 (Must Have)
- [ ] 공간 유형이 명확한가? (원룸/복층/오피스 등)
- [ ] 사용자/용도가 파악되었는가?
- [ ] 스타일 방향이 정해졌는가? (키워드 또는 레퍼런스)

### 권장 확인 (Should Have)
- [ ] 레퍼런스 이미지가 있다면 분석했는가?
- [ ] 싫어하는 요소도 파악했는가?
- [ ] 감정적 목표(어떤 기분을 느끼고 싶은지)를 확인했는가?

### 완료 조건
> 사용자가 "어떤 공간을 어떤 스타일로" 만들고 싶은지 명확하게 설명할 수 있을 때

---

## Phase 2: Planning 체크리스트

### 필수 확인 (Must Have)
- [ ] 주색 (60%)이 결정되었는가?
- [ ] 보조색 (30%)이 결정되었는가?
- [ ] 강조색 (10%)이 결정되었는가?

### 권장 확인 (Should Have)
- [ ] 60-30-10 법칙을 설명했는가?
- [ ] 바닥재가 결정되었는가?
- [ ] 주요 재료/질감 조합이 정해졌는가?

### 완료 조건
> 사용자가 "어떤 색상과 재료로" 만들지 구체적으로 알고 있을 때

---

## Phase 3: Architecture 체크리스트

### 필수 확인 (Must Have)
- [ ] 공간 크기/비율이 정해졌는가?
- [ ] 필요한 영역들이 나열되었는가?
- [ ] 영역 간 관계(오픈/분리)가 결정되었는가?

### 권장 확인 (Should Have)
- [ ] 복층이라면 복층 비율과 용도가 정해졌는가?
- [ ] 동선 흐름을 고려했는가?
- [ ] 시각적 초점(focal point)이 정해졌는가?
- [ ] z-order 계획이 있는가?

### 완료 조건
> 사용자가 "어디에 무엇이 있을지" 머릿속에 그릴 수 있을 때

---

## Phase 4: Creation 체크리스트

### 각 단계별 확인
1. **기본 구조**
   - [ ] 바닥이 생성되었는가?
   - [ ] 벽이 생성되었는가?
   - [ ] 복층/계단(있다면)이 생성되었는가?
   - [ ] 사용자 피드백을 받았는가?

2. **대형 가구**
   - [ ] 위치가 사용자와 함께 결정되었는가?
   - [ ] 배치 후 전체 균형을 확인했는가?

3. **디테일**
   - [ ] 소품이 과하지 않은가?
   - [ ] 전체적인 느낌이 처음 비전과 맞는가?

### 완료 조건
> 사용자가 "이 정도면 됐다"고 만족할 때

---

## 대화 품질 체크리스트

### 피해야 할 것
- [ ] 일방적인 질문 폭격을 하지 않았는가?
- [ ] 예/아니오로만 답할 수 있는 질문만 하지 않았는가?
- [ ] 사용자 의견을 무시하고 진행하지 않았는가?
- [ ] 너무 많은 옵션을 한 번에 제시하지 않았는가?

### 해야 할 것
- [ ] 사용자 답변에 따른 후속 질문을 했는가?
- [ ] 전문 지식을 자연스럽게 공유했는가?
- [ ] 시각적 피드백(캡처)을 자주 보여줬는가?
- [ ] 중요한 결정을 기록(mama_save)했는가?

---

## 최종 검증

### 결과물 확인
- [ ] 처음 비전과 일치하는가?
- [ ] 색상 팔레트가 적용되었는가?
- [ ] 공간 구조가 계획대로인가?
- [ ] 사용자가 만족하는가?

### 기록 확인
- [ ] 주요 결정들이 mama에 저장되었는가?
- [ ] 산출물(artifact)이 저장되었는가?
